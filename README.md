# ConvolutionApp
離散信号の畳み込みの計算を行うプログラム<br>
グラフ出力にも対応<br>
<br>
あらゆる信号の演算が可能<br>
信号の長さや、信号が始まるnの場所は自由<br>
<br>
信号の値も自由自在<br>
マイナスの値や、小数点付きの値でも計算可能<br>
<br>
Normalize機能搭載<br>
無駄な0が続く信号のデータ量を少なくできる<br>
<br>
